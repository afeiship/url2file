@jswork/url2file

url2file

Make url to local file.

installation

 
npm install @jswork/url2file

usage

 
import url2file from '@jswork/url2file';

url2file([
  'https://fasimi.com/',
  'https://fasimi.com/filters/series.html',
  'https://fasimi.com/filters/movie.html'
], {dst:'.'});
